## Numbering convention

- `SYS-Fxxx` — Functional requirements
- `SYS-Pxxx` — Performance
- `SYS-Axxx` — Availability / Reliability
- `SYS-Sxxx` — Safety
- `SYS-Cxxx` — Cybersecurity
- `SYS-Mxxx` — Maintainability
- `SYS-Ixxx` — Interfaces
